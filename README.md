## Run Locally

Clone the project

```bash
  git clone https://github.com/dropecosta/marvel-app.git
```

Go to the project directory

```bash
  cd marvel-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
